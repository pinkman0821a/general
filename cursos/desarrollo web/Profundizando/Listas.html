Listas

Listas "li"

En el desarrollo web moderno, las listas son elementos muy útiles.
Incluso puedes usar listas para crear menús de navegació

Una lista consiste en elementos de lista "li".>

Ejemplo

  • elemento 1
  • elemento 2
  • elemento 3

    • los elementos de lista son tiquetas contenedoras

    Listas "ol"

    Las listas ordenadas "ol" se muestran con números (1, 2, 3...) en lugar de símbolos de viñetas. Usa "ol" cuando los puntos tengan un cierto orden, como instrucciones paso a paso. Completa el código para crear una lista ordenada.

    Ejemplo

    1. elemento 1
    2. elemento 2
    3. elemento 3

    Listas "ul"

    Usa listas no ordenadas "ul" cuando el orden de los elementos no sea importante. Se muestran con viñetas.

    Ejemplo

    • elemento 1
    • elemento 2
    • elemento 3
