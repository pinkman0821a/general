HiperTexto

La Web no sería posible sin el HiperTexto. HiperTexto es lo que permite a los usuarios saltar de una página web a otra a través de enlaces.
En esta lección, aprenderás a conectar páginas web con hipervínculos.

Los hipervínculos permiten a los usuarios:

1. moverse de una página web a otra
2. compartir información entre diferentes máquinas y sistemas

etiqueta "a"

La etiqueta de anclaje "a" se utiliza para crear un hipervínculo en una página web.
La etiqueta "a" es una etiqueta contenedora.

Los hipervínculos se utilizan para enlazar una página web con otras. Para crear un enlace, necesitas "href" para agregar la URL de destino.
href significa referencia HiperTexto.

La URL debe estar encerrada entre comillas para funcionar sin errores. Si olvidas agregar href o las comillas, la etiqueta "a" no creará un hipervínculo.
